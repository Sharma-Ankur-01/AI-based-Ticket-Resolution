# AI-based-Ticket-Resolution

This project poses itself as the foundation of any advanced AI based Ticket Resolution system, Proving an extensive use of libraries in python like Langchain, Streamlit and FastAPI, which can be coupled with any library and development languages to incoporate needs and usage by any firm/individual.

### Introduction
~ Smart Support & Ticket Resolution is an Al-driven system for handling customer support queries.  
~ Automates ticket classification, prioritization, and resolution.  
~ Runs on llama3.2:1b model, providing an easy and feasible making it localised and fast.  
~ Designed to integrate with existing support workflows.  

### Key Concepts used:
~ Natural Language Understanding for ticket interpretation.  
~ Intent classification and entity extraction.  
~ Prompt based article filtering and options.  
~ Vector Search using FAISS from database.  

### Further usage:
~ Customer support automation for Saas platforms.  
~ E-commerce order and refund support.  
~ Telecom, banking, and enterprise support systems.  
~ IT service management (ITSM) and internal helpdesks.
